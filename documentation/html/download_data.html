Data download¶

Documentation of the code in src/download_data.

Download EURO/USD exchange rate data¶

Downloads the daily EURO/USD exchange rate from FRED and saves output as a pickle. url: https://fred.stlouisfed.org/series/DEXUSEU

Descriptive analysis of the Time series¶

plot_historical_timeseries(euro_usd_pd, path)[source]¶

Plot the historical EURO / USD exchange rate and percentage returns

Parameters

  • euro_usd_pd (pd.DataFrame) – daily EURO / USD data (must include price, pct_change and date as index).

  • path (string) – path to output graph.
