********************
Swap Contract/Simulation configurations
********************

The directory *src.model_specs* contains `JSON <http://www.json.org/>`_ files with model specifications.

There are 3 json files in the directory. |br|
Macroeconomic conditions are specified in *scenario_config*. |br|
Simulation parameters are specified in *simulation_config*. |br|
Parameters of the swap contract are specified in *swap_config*. |br|
